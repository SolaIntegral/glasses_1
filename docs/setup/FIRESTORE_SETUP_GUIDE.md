# Firestoreで管理者アカウントを作成する手順

## 画面操作の詳細

### ステップ1: コレクションを作成

1. Firestore Console（https://console.firebase.google.com/project/glasses1-582eb/firestore/databases/-default-/data）にアクセス
2. 「**+ コレクションを開始**」ボタンをクリック
3. コレクションIDに **`users`** と入力
4. 「**次へ**」ボタンをクリック

### ステップ2: ドキュメントIDを設定

5. ドキュメントIDフィールドに **`admin001`** と入力
6. ドキュメントのフィールドを設定（以下5つのフィールドを追加）

### ステップ3: フィールドを追加

#### フィールド1: displayName

7. 「**フィールドを追加**」をクリック
8. フィールド名に **`displayName`** を入力
9. フィールドタイプを「**文字列**」に設定
10. 値に **`管理者`** を入力
11. 「**完了**」をクリック

#### フィールド2: role

12. 「**フィールドを追加**」をクリック
13. フィールド名に **`role`** を入力
14. フィールドタイプを「**文字列**」に設定
15. 値に **`admin`** を入力
16. 「**完了**」をクリック

#### フィールド3: hashedPassword

17. 「**フィールドを追加**」をクリック
18. フィールド名に **`hashedPassword`** を入力
19. フィールドタイプを「**文字列**」に設定
20. 値に以下をコピー&ペースト:
   ```
   cGFzc3dvcmQxMjM=
   ```
21. 「**完了**」をクリック

#### フィールド4: createdAt

22. 「**フィールドを追加**」をクリック
23. フィールド名に **`createdAt`** を入力
24. フィールドタイプを「**タイムスタンプ**」に設定
25. 値の右側にある「**設定**」ボタンをクリック
26. 現在の日時が自動的に設定されることを確認
27. 「**完了**」をクリック

#### フィールド5: updatedAt

28. 「**フィールドを追加**」をクリック
29. フィールド名に **`updatedAt`** を入力
30. フィールドタイプを「**タイムスタンプ**」に設定
31. 値の右側にある「**設定**」ボタンをクリック
32. 現在の日時が自動的に設定されることを確認
33. 「**完了**」をクリック

### ステップ4: 保存

34. 5つのフィールドがすべて追加されていることを確認
35. 「**保存**」ボタンをクリック

### 完成したドキュメントの例

以下のようになっていれば正しいです：

```
users/admin001:
  displayName: "管理者"
  role: "admin"
  hashedPassword: "cGFzc3dvcmQxMjM="
  createdAt: [タイムスタンプ]
  updatedAt: [タイムスタンプ]
```

## ログイン

36. https://glasses1-582eb.web.app にアクセス
37. 「ログイン」をクリック
38. 以下の情報を入力：
   - ユーザーID: `admin001`
   - パスワード: `password123`

## トラブルシューティング

### フィールドが追加できない
- フィールドタイプが正しいか確認（特に`hashedPassword`は「文字列」タイプ）
- 一度保存してから再度編集してください

### ログインできない
- ドキュメントIDが `admin001` になっているか確認
- `role`フィールドの値が `admin` になっているか確認
- `hashedPassword`フィールドの値が正確に `cGFzc3dvcmQxMjM=` になっているか確認
- ブラウザをリロードして再度ログインしてみてください

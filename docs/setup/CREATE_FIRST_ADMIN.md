# 初回管理者アカウント作成手順

## ステップ1: コレクションを作成

1. Firestore Consoleの画面で「+ コレクションを開始」をクリック
2. コレクションIDに `users` と入力して「次へ」をクリック

## ステップ2: ドキュメントを作成

3. ドキュメントIDに `admin001` と入力
4. 以下のフィールドを追加：

### フィールド1: displayName
- フィールド名: `displayName`
- フィールドタイプ: `文字列`
- 値: `管理者`
- 「完了」をクリック

### フィールド2: role
- フィールド名: `role`
- フィールドタイプ: `文字列`
- 値: `admin`
- 「完了」をクリック

### フィールド3: hashedPassword
- フィールド名: `hashedPassword`
- フィールドタイプ: `文字列`
- 値: `cGFzc3dvcmQxMjM=`
- 「完了」をクリック

### フィールド4: createdAt
- フィールド名: `createdAt`
- フィールドタイプ: `タイムスタンプ`
- 値: 現在の日時（「設定」ボタンをクリックして選択）
- 「完了」をクリック

### フィールド5: updatedAt
- フィールド名: `updatedAt`
- フィールドタイプ: `タイムスタンプ`
- 値: 現在の日時（「設定」ボタンをクリックして選択）
- 「完了」をクリック

5. 「保存」をクリック

## ステップ3: ログイン

6. https://glasses1-582eb.web.app にアクセス
7. 「ログイン」をクリック
8. 以下の情報を入力：
   - **ユーザーID**: `admin001`
   - **パスワード**: `password123`
9. 管理者ダッシュボードが表示されれば完了です！

## トラブルシューティング

### ドキュメントが作成できない
- フィールドタイプが正しいか確認してください
- 全てのフィールドを追加してから「保存」をクリックしてください

### ログインできない
- `role`フィールドが `admin` になっているか確認
- `hashedPassword`フィールドが `cGFzc3dvcmQxMjM=` になっているか確認
- ブラウザをリロードして再度ログインしてみてください

## 次のステップ

管理者としてログインできたら：
1. 生徒アカウントを作成: 管理画面 > 生徒管理 > 新規アカウント発行
2. 講師アカウントを作成: 管理画面 > 講師管理 > 新規アカウント発行
